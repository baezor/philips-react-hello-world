# Hello World React Philips
> Getting started with Phillips Hue API.

## Setup project

### Requirements:
1. Philips Hub
2. API of your bridge
3. Node and NPM
4. Caffeine

### Installation

1. Install dependencies:

```
yarn install

```

2. Build project

```
yarn dev

```

3. Open [http://localhost:1234](http://localhost:1234) you should see the server up and running. 

4. Happy Hacking!
